Get Started with HP Printer - 123.hp.com/setup


  • Step 1: Unbox and Prepare
    • Unbox the Printer: Carefully unpack your printer and remove all protective materials.
    • Connect Power and Turn On: Plug in the power cord and turn on the printer.
  • Step 2: Install Ink Cartridges and Paper
    • Load Paper: Open the paper tray and load plain paper.
    • Install Ink Cartridges: Follow the printer manual to insert ink cartridges. Make sure they click into place.
  • Step 3: Software Installation
    • Visit 123.hp.com/setup: Open your web browser and go to 123.hp.com/setup.
    • Enter Your Printer Model: Type your printer model (e.g., HP Envy 4500) and click “Search.”
    • Download Software: Download the recommended software and drivers for your printer.
  • Step 4: Network Setup
    • Connect to Wi-Fi: Use the printer’s control panel to connect to your Wi-Fi network.
    • Test Printing: Print a test page to ensure everything is working correctly.
  • Step 5: Additional Features
    • Mobile Printing: Set up mobile printing using the HP Smart app or Google Cloud Print.
    • Scan and Copy: Explore scanning and copying features based on your printer model.

Download and Install HP Printer Drivers


  • Switch on the HP printer and computer.
  • Open web browser and type 123.hp.com/setup.
  • Make sure to install cartridges, load paper, and power on the printer. Click Download to get the recommended software for your printer.
  • Enter HP Printer model number in the space provided.
  • Choose the printer driver.
  • Click on "download" button.
  • Go to the downloaded folder and run the software file to install the drivers.

HP Printer Wired Setup via USB Cable


  • If it is connected already, remove the cable, then search "devices and printers" from control panel settings and right-click on the hp icon.
  • Click Remove device, then open 123.hp.com/setup URL, or if you have Laserjet printer open 123.hp.com/laserjet link.
  • Click Download, choose setup, and select your device type (if it doesn't detect your device); wait for hp printer setup download.
  • Double-click on the software you downloaded from 123.hp.com/setup and choose USB connection when prompt.
  • Follow on-screen instructions and connect one port of USB cable to PC and the other to printer port once installed.
  • Now, HP Printer via USB cable from 123.hp.com/setup is set up; try print, scan, or fax.
